
This is our Agent Harness configuration. To use it, clone it to `~/.config/agents` and symlink to your favourite harness.

# Example commands for using unravel-team/agents configuration with the Pi Coding Agent

1. `git clone https://github.com/unravel-team/agents ~/.config/agents`
2. `ln -s ~/.pi/agent/agents ~/.config/agents/agents`
3. `ln -s ~/.pi/agent/prompts ~/.config/agents/prompts`
4. `ln -s ~/.pi/agent/AGENTS.md ~/.config/agents/AGENTS.md`
5. `ln -s ~/.pi/agent/APPEND_SYSTEM.md ~/.config/agents/APPEND_SYSTEM.md`
6. `ln -s ~/.pi/agent/extensions ~/.config/agents/pi-extensions`
7. `ln -s ~/.pi/agent/skills ~/.config/agents/skills`
